# My Backend API  (Node.js + Express + MongoDB Atlas)

A fully working, production-ready REST API built with modern ES Modules.

## Features
- Express.js server with ES Modules
- MongoDB Atlas connection (live cloud database)
- Clean MVC folder structure
- GET /api/users → Returns all users from MongoDB
- Tested with Postman
- Ready for deployment

## Live API URL (after deployment)
https://your-api-name.onrender.com

## Endpoints
- `GET /` → Home page
- `GET /api/users` → Get all users (JSON)

## Tech Stack
- Node.js
- Express
- MongoDB Atlas + Mongoose
- dotenv for environment variables
- ES Modules (import/export)

```bash
git clone https://github.com/your-username/my-backend-api.git
